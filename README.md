# NBOBanking-Backend
The backend service for a banking application called No Better Option Banking. This will handle any HTTP calls from the frontend service and return the requested data as long as nothing is wrong with the call.

## Features
- I can login or register an account for signing in
- I can create a persisting bank account (Checking/Savings or both)
- I can create a credit account that will have charges needed to be paid off over time
- I will be able to deposit and withdraw money
- I can see the predicted interest charge for any credit accounts
- I can change personal info and password
- I can see a purchase history/bank statement
